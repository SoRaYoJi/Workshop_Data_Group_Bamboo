# SCI19 3431: Workshop in Data Intelligence and Visualization

This folder structure is designed for the course project.

## 1. Project_Info
- ข้อมูลรายวิชา รายละเอียดโครงการ อาจารย์ผู้สอน

## 2. Data
- Raw_Data: ข้อมูลดิบ
- Processed_Data: ข้อมูลที่ผ่านการทำ preprocessing
- Visualization_Data: ข้อมูลที่ใช้ทำ visualization

## 3. Code
- Notebooks: Jupyter Notebooks สำหรับการทดลอง
- Scripts: Python scripts (.py) สำหรับ pipeline

## 4. Models
- Trained_Models: โมเดลที่ train แล้ว
- Experiments: การทดลองและผลลัพธ์

## 5. Reports
- Drafts: รายงานร่าง
- Final: รายงานฉบับสมบูรณ์

## 6. Presentations
- Slides: สไลด์นำเสนอ
- Demo: วิดีโอสาธิตหรือ demo files

## 7. Submission
- ไฟล์ที่ต้องส่งอาจารย์ เช่น รายงานสุดท้าย โค้ด โมเดล
